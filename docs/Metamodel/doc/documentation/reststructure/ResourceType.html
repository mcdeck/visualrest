Classifier reststructure.ResourceType

Package List

<<abstract>>
the abstract classifier to define a resource type
Attribute: name : EString
the name of this ResourceType
Attribute: isSingleton : EBoolean (default: false)
true if this resource type defines a singleton
Attribute: createOnStart : EBoolean (default: false)
true if the instance of this resource type is automatically created during the start of the application, only allowed when isSingleton is true
Reference: identifer : ResourceIdentifierPattern [ 1..* ]
the identifier patterns for this ResourceType
Reference: resourceelements : ResourceElement [ * ]
the elements which describe the properties of this ResourceType
Reference: methods : Method [ * ]
the methods which describe the behavior in the different states of this ResourceType
Reference: creators : Creator [ * ]
the creators which enable the creation of new resources of this type
Reference: states : State [ 1..* ]
the states of this ResourceType
Reference: initialState : State [ 1 ]
the initial state of this ResourceType
